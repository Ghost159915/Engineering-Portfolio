Role Gen AI Assisted Programming
Year 2025
Stack HTML, CSS, JavaScript

Overview

In this capstone project, I partnered with ChatGPT to architect, prototype, and refine my personal engineering portfolio site. Leveraging conversational prompts, I sketched the site map, iterated on responsive layouts using CSS flex and grid, and honed interactive components such as countdown timers, image galleries, and GitHub embed sections. AI guidance streamlined critical decisions, suggested semantic HTML structures, and provided concise copy for project overviews.

Key screenshot

How it was built

  1. Outlined requirements for the About, Projects, and Contact sections.
  2. Crafted precise prompts for semantic HTML, responsive CSS, and JS.
  3. Iterated on layouts, typography, and content for clarity.
  4. Committed changes to GitHub using version control workflows.
  5. Deployed with Netlify using a custom domain and SSL.

While AI wrote the core code, I reviewed and adjusted every snippet, quickly mastering web fundamentals. This rapid, AI-driven workflow delivered a polished portfolio in one day and is a model I will apply to future projects.
